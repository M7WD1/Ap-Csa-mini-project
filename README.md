# Ap-Csa-mini-project
 
This is a Java program that uses a grid (2D array).  
The user enters numbers, and the program can do many operations on the grid.

What can it do?

- Show the grid
- Find row and column sums
- Find biggest and smallest number
- Count how many times a number appears
- Check if a row is increasing
- Rotate a row
- Swap two rows
- Reverse a column
- Work on a small part of the grid (subgrid)
- Print boundary and diagonals
- Check if there are duplicates in a row

---

The way it works: 

1. You enter:
   - number of rows
   - number of columns
   - the numbers inside the grid

2. A menu will appear

3. You choose what you want to do by typing a number

4. The program will show the result
